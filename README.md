# HJRotatoView
一个实用的3D旋转动画
##我并不是什么大神，这是我的第一个开源作品，业余写的一个小小的3D动画，希望可以得到大家的支持！！
###演示
![](https://github.com/JueJueHuang/HJRotatoView/raw/master/HJRotato.gif)

###使用方法
```Swift
//将两个view传入下面的构造方法即可
    lazy var fronV:HJRotatoView = HJRotatoView(fristV: "第一个view", secondV: "第二个view")
```
通过左/右滑动即可出现翻转效果
